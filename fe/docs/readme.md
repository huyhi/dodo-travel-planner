## 📁 前端项目结构
```
proj/
├── 📁 fe/                    # 前端应用
│   ├── 📁 app/
│   │   ├── 📁 components/    # React 组件
│   │   ├── 📁 services/      # API 服务
│   │   ├── 📁 models/        # 类型定义
│   │   └── 📁 styling/       # 样式文件
└── └── 📄 package.json       # Node.js 依赖
```

## Frontend Setup

### A. 安装 NodeJS

[node link](https://nodejs.org/en/download/)

### B. 安装依赖 & 启动服务

```bash
# 📱 进入前端目录
cd fe

# 📦 安装依赖
npm install

# 🚀 启动开发服务器
npm run dev
```

### C. 打开浏览器访问

[http://localhost:3000](http://localhost:3000)

如果可以正常访问则代表前端服务准备就绪
